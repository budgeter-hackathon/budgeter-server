# Budgeter Server-Side Express Application

## Routes

```
Endpoint              | HTTP Method | Result                           
----------------------+-------------+------------------------------------
/api/transactions     | GET         | Returns all transactions         
/api/transactions/:id | GET         | Returns single transaction by id 
----------------------+-------------+------------------------------------
/api/categories       | GET         | Returns all categories           
/api/categories/:id   | GET         | Returns single category by id    
----------------------+-------------+------------------------------------
/api/accounts         | GET         | Returns all accounts             
/api/accounts/:id     | GET         | Returns account by id            
----------------------+-------------+------------------------------------      
```
